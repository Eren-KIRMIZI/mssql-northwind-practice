# MSSQL Northwind Practice

This repository is a comprehensive Microsoft SQL Server (MSSQL) practice project built on the classic **Northwind** sample database.

The goal of this repository is to cover SQL concepts **from fundamentals to advanced level**, using realistic business scenarios and production-style query patterns.

---

### 1. SELECT Queries
- Basic SELECT statements
- WHERE conditions
- LIKE, BETWEEN, IN
- ORDER BY, TOP, DISTINCT
- Aliases and expressions
- String and date functions
- CASE WHEN
- Subqueries
- EXISTS
- UNION / UNION ALL

### 2. JOIN Operations
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN
- SELF JOIN
- Multiple table joins

### 3. GROUP BY & HAVING
- Aggregations
- HAVING conditions
- GROUP BY with JOINs

### 4. Views
- Sales summary views
- Product-category views
- Customer-sales views

### 5. Stored Procedures
- Parameterized procedures
- Monthly sales reports
- Stock status procedures

### 6. User Defined Functions
- Scalar functions
- Date difference functions
- Total sales calculations

### 7. Triggers
- INSERT triggers
- Stock control triggers
- DELETE logging triggers

### 8. Transactions
- Stock and sales transactions
- Transfer transactions
- Rollback scenarios

### 9. Indexes
- Single-column indexes
- Date indexes
- Composite indexes
- Advanced indexing strategies

### 10. DML Operations
- INSERT
- UPDATE
- DELETE
- MERGE

### 11. Error Handling
- TRY...CATCH basics
- Transactions with error handling
- Error logging

### 12. Performance
- Execution plan analysis
- SET STATISTICS IO / TIME
- Index vs non-index performance comparison

### 13. Temporary Tables
- Temporary tables
- Table variables
- Performance comparison

### 14. Common Table Expressions (CTE)
- Basic CTE usage
- CTE with JOINs
- Recursive CTEs

### 15. Window Functions

- Running totals
